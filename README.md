# Guía Manual de Programación en PLAXIS 2D

Este repositorio contiene el desarrollo del código empleado para la modelación numérica de excavaciones con muros de contención y anclajes en el programa PLAXIS 2D, utilizando la interfaz de scripting en Python.
El propósito de este material es proporcionar un recurso de referencia clara y práctica para reproducir la creación y configuración de modelos directamente desde el entorno Input de PLAXIS 2D.

## Descripción
El código implementa la generación automatizada de los principales elementos del modelo, incluyendo la definición de materiales, geometrías, muros de contención, anclajes y fases de construcción.  
Asimismo, permite realizar el control de los cálculos y la activación y desactivación de fases, siguiendo un flujo de trabajo ordenado y replicable.
